# Task 2 - Data Visualization and Storytelling

## Objective
Create an interactive Power BI dashboard using the Superstore dataset to present business insights.

## Dashboard Includes
- KPI Cards (Sales, Profit, Orders, Quantity)
- Sales by Category
- Profit by Sub-Category
- Sales by Region
- Ship Mode Analysis
- Monthly Sales Trend
- Top Customers
- Interactive Slicers (Category, Region, Segment, Year)

## Business Insights
- Technology has the highest sales.
- Copiers are the most profitable sub-category.
- West region generates the highest sales.
- Standard Class is the most preferred shipping mode.
- Sales show seasonal fluctuations.

## Tool Used
Microsoft Power BI Desktop
